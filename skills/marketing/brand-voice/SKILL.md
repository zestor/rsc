# Brand Voice

## Voice Attribute Definition

For each chosen attribute (pick 3-5), document:

- **We are**: what this means in practice
- **We are not**: common misinterpretation to avoid
- **This sounds like**: example sentence demonstrating the attribute
- **This does NOT sound like**: example sentence violating the attribute

### Common Attribute Spectrums

| Spectrum   | One End               | Other End                 |
| ---------- | --------------------- | ------------------------- |
| Formality  | Formal, institutional | Casual, conversational    |
| Authority  | Expert, authoritative | Peer-level, collaborative |
| Emotion    | Warm, empathetic      | Direct, matter-of-fact    |
| Complexity | Technical, precise    | Simple, accessible        |
| Energy     | Bold, energetic       | Calm, measured            |
| Humor      | Playful, witty        | Serious, earnest          |

## Tone Adaptation

Voice attributes remain fixed. Tone dials them up or down based on context.

### By Channel

| Channel           | Tone Adaptation                          |
| ----------------- | ---------------------------------------- |
| Blog              | Informative, conversational, educational |
| LinkedIn          | Professional, thought-provoking, concise |
| Twitter/X         | Punchy, direct, sometimes witty          |
| Email marketing   | Personal, helpful, action-oriented       |
| Sales collateral  | Confident, benefit-driven, specific      |
| Support/Help docs | Clear, patient, step-by-step             |
| Press release     | Formal, factual, newsworthy              |
| Error messages    | Empathetic, helpful, blame-free          |

### By Situation

| Situation                              | Tone Adaptation                                 |
| -------------------------------------- | ----------------------------------------------- |
| Product launch                         | Excited, confident, forward-looking             |
| Incident or outage                     | Transparent, empathetic, accountable            |
| Customer success story                 | Celebratory, specific, crediting the customer   |
| Bad news (price increase, deprecation) | Honest, respectful, solution-oriented           |
| Competitive comparison                 | Confident but fair, fact-based, not disparaging |

## Style Guide Rules

Document and enforce these choices consistently:

| Rule              | Options                                       |
| ----------------- | --------------------------------------------- |
| Oxford comma      | Yes / No                                      |
| Heading case      | Sentence / Title                              |
| Contractions      | Use / Avoid                                   |
| Em dash spacing   | No spaces / Spaces                            |
| Numbers           | Spell out 1-9, numerals 10+ / Always numerals |
| Exclamation marks | Limited use, never more than one              |
| Emoji by channel  | Professional: none; social: where appropriate |

## Terminology Management

Maintain preferred/avoided term lists covering:

- **Verb vs. noun forms**: sign up (verb) vs. signup (noun), log in vs. login, set up vs. setup
- **Product names**: official capitalization, full name vs. shorthand, trademark symbol rules
- **Inclusive language**: gender-neutral, person-first, no ableist terms, avoid culturally specific idioms
- **Industry jargon**: which terms the audience knows, which need defining, acronym spell-out rules
- **Competitor terms**: terms competitors coined that you should avoid to prevent reinforcing their positioning

## Gotchas

- **Voice and tone are not the same thing** — voice is the brand's personality (constant), tone is the emotional inflection (varies by context). Conflating them leads to inconsistent or flat copy.
- **"Approachable" does not mean dumbed-down** — most brands that pick "approachable" as an attribute end up losing substance. Define what the attribute is NOT, not just what it is.
- **Style guides rot fast** — if the guide is not enforced during review, it becomes fiction. Attach rules to your content review checklist.
- **Terminology drift across teams** — sales, support, and marketing will independently coin different names for the same feature. A shared glossary prevents customers from seeing three names for one thing.
- **Translating voice across languages is not just translation** — humor, idioms, and formality registers differ by culture. Voice guidelines need per-locale adaptation, not just word-for-word translation.
