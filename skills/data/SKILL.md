- `data/sql-queries` — Write correct, performant SQL across major data warehouse dialects.
- `data/statistical-analysis` — Apply statistical methods: descriptive stats, hypothesis testing, outlier detection.
- `data/validation` — QA an analysis before sharing — methodology checks, accuracy verification, pitfall detection.
