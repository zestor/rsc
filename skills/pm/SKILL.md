# Product Management Skills

- `pm/competitive-analysis` — Analyze competitors with feature comparison matrices, positioning analysis, and win/loss methodology.
- `pm/feature-spec` — Write PRDs with problem statements, user stories, MoSCoW requirements, and success metrics.
- `pm/metrics-tracking` — Define product metrics hierarchies, set OKRs, and design dashboards.
- `pm/roadmap-management` — Prioritize roadmaps using RICE/ICE/MoSCoW, map dependencies, and plan capacity.
- `pm/stakeholder-comms` — Draft audience-tailored updates, communicate risks with ROAM, and document decisions as ADRs.
- `pm/user-research-synthesis` — Synthesize qualitative and quantitative research into themes, personas, and sized opportunities.
