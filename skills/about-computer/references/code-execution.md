# Code Execution and File Workspace

Computer runs code in a sandboxed Linux VM with a persistent file workspace.

## Environment

- **OS**: Sandboxed Linux VM.
- **Pre-installed**: Python, Node.js, ffmpeg, yt-dlp, and standard Unix tools.
- **Package installation**: Install additional packages on the fly as needed.
- **Workspace**: Persistent directory where files are created, read, and managed across the session.

## What it can do

- Write and execute Python or Node scripts in real time, iterating on results.
- Process data files — CSV, JSON, Excel, PDF, images, audio, video.
- Run shell commands for system operations, data processing, and automation.
- Read, write, edit, search, and manage files across the workspace.
- Generate charts, visualizations, and data analysis outputs.

## Example use cases

- Clean and analyze a dataset — load CSV, normalize values, compute statistics, generate charts.
- Process media — convert video formats, extract audio, resize images, generate thumbnails.
- Build automation scripts — parse structured data, transform formats, generate reports.
- Prototype algorithms — implement, test, and refine logic interactively.
- Install and use specialized libraries — pandas, matplotlib, scikit-learn, whatever the task needs.
