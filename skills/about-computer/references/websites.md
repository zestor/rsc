# Live Website Deployment

Computer builds and deploys websites to a public URL — in a single conversation.

Website publishing is unavailable in Incognito Mode because deployment creates a public URL.

## What it can do

- Build HTML/CSS/JS sites with modern frameworks and responsive design.
- Deploy to a live public URL instantly.
- Add server-side logic via backend servers (SQLite databases, form backends, webhooks).
- Edit files and re-deploy to the same URL with no manual deployment steps.

## Three types of websites Computer can build

Computer has specialized expertise in three categories of web development:

### Informational Websites

Personal sites, portfolios, editorial publications, blogs, small business pages, and landing pages. Content-first design with beautiful typography, SEO optimization, and responsive layouts. Think portfolio showcases, company sites, magazine-style editorial pages, and single-page landing pages.

### Web Applications

SaaS products, dashboards, admin panels, e-commerce stores, and interactive brand experiences. Full-featured product interfaces with navigation architecture, state management, data visualization, authentication, and responsive app patterns. Think Linear, Notion, Shopify-style admin panels, and analytics dashboards.

### Web-Based Games

Browser-based games built with Three.js, Babylon.js, WebGL, and modern web game engines. From casual 2D puzzles to immersive 3D worlds — playable instantly in any browser with no install required. Includes asset sourcing from Fab.com, Sketchfab, and other marketplaces.

## Workflow

1. Computer writes the HTML, CSS, and JavaScript in the workspace.
2. Deploys to a public URL.
3. User reviews the live site.
4. Computer makes changes and re-deploys to the same URL.

The cycle from idea to live URL takes minutes. Iteration is instant.
