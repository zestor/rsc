# Prompt Variants Template

Use this when the user asks for multiple directions or variants. Match the requested count when provided.

```md
## Direction
[One sentence describing the shared goal]

## Variants

### [Variant name]
[Closest to the user's request. Strong fidelity, minimal interpretation.]

### [Variant name]
[More stylized or exploratory. Preserve core subject and constraints.]

### [Variant name]
[Most reliable for the target format, crop, and model.]

## Avoid
[Shared constraints only]

## Model
[Model choice and one-sentence reason, if relevant]
```

Variants must differ by a real creative axis: composition, lighting, material, environment, color, camera distance, rendering style, or level of abstraction. Do not create variants by swapping adjectives.

For mixed crops, transparency needs, or mixed reference roles, include model and aspect-ratio notes per variant instead of using only one shared model line.

For 4+ variants, add named creative axes rather than expanding generic slots. Examples: "Close product crop," "Wide editorial environment," "High-key studio," "Low-light cinematic," "Graphic color-blocked."
