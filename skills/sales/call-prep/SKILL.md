# Call Prep

Prepare for any sales call with a structured prep brief.

## Execution Flow

1. Gather context: company, meeting type, attendees (from user or connectors)
2. Pull connector data: calendar, CRM, email, chat, transcripts
3. Web search: recent news, funding, leadership changes, attendee profiles
4. Synthesize into a prep brief with agenda and questions

## Connector Enrichment

| Connector       | What It Adds                                                    |
| --------------- | --------------------------------------------------------------- |
| **Calendar**    | Auto-find meeting, pull attendees and description               |
| **CRM**         | Account details, contact history, open deals, recent activities |
| **Email**       | Recent threads, open questions, commitments made                |
| **Chat**        | Internal discussions, colleague insights about the account      |
| **Transcripts** | Prior call recordings, objections raised, topics covered        |

## Meeting Type Adjustments

| Type               | Agenda Emphasis                      | Key Output                                |
| ------------------ | ------------------------------------ | ----------------------------------------- |
| **Discovery**      | Questions > talking                  | Qualification signals, next step proposal |
| **Demo**           | Tailored examples for their use case | Technical requirements, decision timeline |
| **Negotiation**    | Handle objections, close gaps        | Path to agreement, clear next steps       |
| **QBR / Check-in** | Review wins, surface new needs       | Renewal confidence, upsell opportunities  |

## Prep Brief Structure

1. **Account Snapshot** -- company, industry, size, deal status, last touch
2. **Who You're Meeting** -- per-attendee: background, role in deal, talking point
3. **Context & History** -- prior interactions, open commitments, recent company news
4. **Suggested Agenda** -- 4-5 items tailored to meeting type, ending with next steps
5. **Discovery Questions** -- 3-5 questions targeting gaps in understanding
6. **Potential Objections** -- likely objections with suggested responses

## Gotchas

- **Research attendees individually** -- a VP Engineering and a CFO in the same meeting need different hooks; don't treat the group as monolithic.
- **Stale CRM data kills credibility** -- if CRM says "last contact 6 months ago," acknowledge the gap rather than pretending continuity.
- **Agenda should flex** -- suggest an agenda, but the best reps adapt to what the buyer wants to discuss first.
- **Prior call transcripts are gold** -- reusing the prospect's exact language from a previous call builds trust faster than any framework.
- **Don't over-prep the slide deck** -- the prep brief is for the seller's head, not for screen-sharing with the buyer.
